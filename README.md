# Tmux Config
- boring looking tmux config for personal use case. i dont think this may be useful for anyone else.

#### Installation
- Intall tmux using `apt` or `yay` or whatever you have.
- Clone the config to `~` dir:
```bash
wget https://raw.githubusercontent.com/Rezy-Dev/tmux-config/refs/heads/main/.tmux.conf
```
- Run `tmux` then press `prefix/CTRL + B` + `I` (capital I)

# If you change `.tmux.conf` file, open `tmux` then run:
- `$TMUX_PLUGIN_MANAGER_PATH/tpm/scripts/install_plugins.sh`. Reload tmux. Done!
